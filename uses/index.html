Things I Use

Desktop OS

My work computer runs Pop!_OS primarily to keep the IT department happy. My daily driver is Fedora, which I install on almost every single computer I own.

Laptop

For work I have a System76 Darter Pro that was issued to me for work. I am extremely happy that I somehow landed a job at a Linux-friendly company.

My everyday computer is a Lenovo Thinkpad x13. I purchased it used and refurbished it. Unfortunately, part of the memory is corrupted and soldered to the main board. Thanks to Linux, I have disabled the problematic section. I love reusing "trash" electronics, so I have an unreasonable amount of affection for this machine.

Text Editor

I use Neovim, and I prefer to run Lunar Vim instead of my own custom config. Lunar Vim gives me everything I need in a code editor out of the box. I make very minimal adjustments to the default Lunar Vim install.
